2017-05-02

Goal if this vignette is to measure the run-time speed of calc_act, for RBeast issue #10.

library(RBeast)

Create a trace:

set.seed(42)
trace <- runif(n = 1600, min = 0.0, max = 1.0)

Profiling single function:

if (1 == 2) {
rprof_tmp_output <- "~/tmp_RBeast_rprof"
Rprof(rprof_tmp_output)
calc_act(trace = trace, sample_interval = 1)
#calc_act_alt(trace = trace, sample_interval = 1)
Rprof(NULL)
summaryRprof(rprof_tmp_output)
}
